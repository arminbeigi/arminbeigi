روی محصولات متغیر ووکامرس یک پیام هشدار درباره‌ی قیمت مدل‌ها نمایش می‌دهد.

== Description ==

برای محصولاتی که چند مدل با قیمت‌های متفاوت دارند (مثل دیگ سوپر ۴۰۰ با ۵ تا ۱۳ پره)،
این افزونه یک پیام هشدار دقیقاً بالای کادر انتخاب مدل/پره نمایش می‌دهد تا مشتری قیمت
یک مدل را با مدل دیگر اشتباه نگیرد. این افزونه به هدر، فوتر، نوار پایین موبایل و دکمه
«افزودن به سبد» دست نمی‌زند.

ویژگی‌ها:
* نمایش خودکار فقط روی محصولات متغیر (روی محصولات ساده اثری ندارد).
* فعال روی همه‌ی محصولات متغیر فعلی و آینده، بدون نیاز به ویرایش تک‌تک محصولات.
* صفحه‌ی تنظیمات برای ویرایش عنوان، متن و رنگ هشدار بدون کدنویسی.
* استایل سبک و واکنش‌گرا (موبایل‌پسند) و هماهنگ با راست‌چین.

== Installation ==

۱. فایل zip افزونه را از مسیر «افزونه‌ها → افزودن → بارگذاری افزونه» در ووردپرس آپلود کنید.
۲. افزونه را فعال کنید.
۳. (اختیاری) از «تنظیمات → هشدار قیمت متغیر» متن و رنگ پیام را ویرایش کنید.

== Frequently Asked Questions ==

= روی محصولات ساده هم نمایش داده می‌شود؟ =
خیر. فقط روی محصولات «متغیر» ووکامرس نمایش داده می‌شود.

= متن پیام را چطور تغییر بدهم؟ =
از منوی «تنظیمات → هشدار قیمت متغیر» عنوان و متن را ویرایش کنید.
