## مشخصات کلی برای تمام مدل‌ها

| مشخصه | مقدار |
|---------|---------|
| **نوع** | مبدل صفحه‌ای براژ‌شده (Brazed Plate Heat Exchanger) |
| **ماده** | استنلس استیل 316 (Stainless Steel 316) |
| **نوع سیال** | آب به آب (Water-to-Water) |
| **فشار کاری دائم** | تا ۳۰ bar |
| **فشار تست** | ۴۵ bar |
| **دمای کاری** | -۱۹۶°C تا ۲۰۰°C |
| **راندمان انتقال حرارت** | بیش از ۹۵% |
| **ضریب انتقال حرارت** | ۵۰۰۰ w/m²°C |
| **صرفه‌جویی انرژی** | تا ۴۰% |
| **نوع اتصالات** | B1، B2 (آب داغ)، S1، S2 (آب سرد) |
| **جریان** | موازی و مخالف (Counter-flow) |
