# 🛒 CRO Strategy — shofazh.com

**Current CRO score: 56/100 (D+).** This is the **single biggest commercial upside** on the site. The store has strong traffic‑earning content but a weak conversion machine. Iranian HVAC buying is **phone/relationship‑led + high‑consideration** — the strategy below is built for that reality.

## Conversion Leaks (prioritised)
| # | Leak | Severity | Fix | Est. Impact |
|---|---|---|---|---|
| 1 | Scattered above‑the‑fold info (price/stock/warranty/CTA) | 🔴 | **Sticky buy‑box** consolidating all decision elements | High |
| 2 | No visible social proof / reviews | 🔴 | Collect & display real reviews, ratings, install photos | High |
| 3 | Phone/WhatsApp not prominent | 🔴 | Persistent click‑to‑call + WhatsApp (esp. mobile bottom bar) | High |
| 4 | Official‑dealer & warranty trust not at decision point | 🟠 | Trust‑badge cluster in buy‑box + cart | Med‑High |
| 5 | No clear UVP above fold | 🟠 | Sitewide value band: genuine warranty · fast dispatch · expert advice | Med |
| 6 | Calculator doesn't capture leads | 🟠 | "نتیجه را برایم بفرست" email/SMS capture | Med |
| 7 | No urgency/scarcity cues | 🟡 | Real stock count, "price valid until", seasonal nudges (ethical only) | Med |
| 8 | Checkout friction (unverified) | 🟠 | Guest checkout, minimal fields, trusted gateway, clear shipping/return | High |

## Psychological Triggers (ethical)
- **Authority:** "نماینده رسمی" badges + expert guides → trust transfer.
- **Social proof:** review counts, "X نفر این ماه خریدند", install photos.
- **Reciprocity:** free capacity calculator + buying guides → goodwill → lead.
- **Commitment:** calculator result → tailored product recommendation → cart.
- **Loss aversion:** honest stock/price‑validity cues.

## Trust System
| Placement | Element |
|---|---|
| Header | Phone + "نماینده رسمی" + years in business |
| Product buy‑box | Warranty + availability + dealer badge + reviews |
| Cart/checkout | Secure‑payment + return policy + support number |
| Footer | About, contact, address, social, namad/enamad trust seals (Iranian e‑commerce trust marks) |

## A/B Test Backlog
1. Sticky buy‑box vs. control → **primary metric: add‑to‑cart + call clicks**.
2. Click‑to‑call prominence (bottom bar vs. inline).
3. Reviews block above vs. below specs.
4. UVP value band on/off.
5. Calculator CTA placement on category pages.

## KPIs to Instrument (GA4)
- Add‑to‑cart rate, begin‑checkout, purchase rate.
- `click_to_call`, `whatsapp_click`, `calculator_complete`, `calculator_lead`.
- Product‑page → cart conversion by category/brand.
- Mobile vs. desktop conversion gap (expect mobile lift after §performance fixes).
