# افزونه‌ی یارا — میزبان پیش‌فاکتور (File Store)

این افزونه «پشت‌صحنه» است: نرم‌افزار دسکتاپ یارا فایل پیش‌فاکتور را روی
هاست شما آپلود می‌کند و یک **لینک معنادار و قابل‌اعتماد** می‌سازد
(`yourdomain.com/factor/shofazh-1024-k7m9` = نام فروشنده + شماره فاکتور +
کد کوتاهِ غیرقابل‌حدس). سپس همان لینک با **پنل پیامکِ خودِ مشتری** برای
گیرنده ارسال می‌شود. کاربر نهایی این بخش را نمی‌بیند.

## نصب
1. پوشه‌ی `yara-filestore` را در `wp-content/plugins/` بگذارید
   (یا از زیپ همین پوشه: پیشخان وردپرس > افزونه‌ها > افزودن > بارگذاری).
2. افزونه را **فعال** کنید.
3. **مهم:** به تنظیمات > پیوندهای یکتا (Permalinks) بروید و فقط دکمه‌ی
   «ذخیره» را بزنید تا قانون بازنویسی مسیر `/factor/` ثبت شود.

## بررسی
- یک فایل تستی از نرم‌افزار بفرستید؛ در پیشخان منوی
  «پیش‌فاکتورهای یارا» باید لینک ساخته‌شده را ببینید.
- باز کردن لینک `/factor/{slug}` باید PDF را نشان دهد.

## احراز هویت
- اگر روی سایت سیستم لایسنس (Digital License Manager یا LMFWC) نصب باشد،
  آپلود فقط با **کلید لایسنس فعال** مجاز است.
- اگر سیستم لایسنسی نصب نباشد (نسخه‌ی شخصی)، آپلود آزاد است.

## محل ذخیره‌ی فایل‌ها
`wp-content/uploads/yara-files/` — با `.htaccess` از فهرست‌شدن محافظت شده
و فقط از طریق مسیر `/f/{token}` سرو می‌شود.
