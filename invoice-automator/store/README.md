# فروشگاه و لایسنس کارا

## فایل‌ها
- `landing.html` — صفحه‌ی فروش آماده (راست‌چین، برند کارا). می‌توانی مستقیم میزبانی کنی
  یا محتوایش را در یک برگه‌ی وردپرس (با Elementor → ویجت HTML) قرار دهی.
- `kara-license-endpoint.php` — افزونه‌ی فعال‌سازی لایسنس برای وردپرس.

## راه‌اندازی فروشگاه (وردپرس + ووکامرس)
1. **WooCommerce** را نصب کن.
2. افزونه‌ی **License Manager for WooCommerce** (رایگان) را نصب کن.
3. برای هر پلن یک **محصول** بساز (نوع: مجازی + قابل‌دانلود):
   - فایل دانلود = نصاب کارا (`InvoiceAutomator-Setup.exe`).
   - در تب License Manager محصول، «تولید خودکار کلید» را فعال کن و
     **Times Activated (max)** را برابر تعداد دستگاه مجاز بگذار (مثلاً ۱).
4. **درگاه پرداخت** را وصل کن (زرین‌پال/زیبال/آی‌دی‌پی). نیاز به **اینماد**.
5. فایل `kara-license-endpoint.php` را در `wp-content/mu-plugins/` بگذار
   (اگر پوشه نبود بساز). در صورت نیاز `kara_plan_name()` را با شناسه‌ی محصولات ویرایش کن.

## اتصال نرم‌افزار به فروشگاه
در فایل `gui/license.py` مقدار `STORE_URL` را به آدرس فروشگاهت تغییر بده،
سپس نصاب را دوباره بساز:
```bash
bash windows-installer/build_on_linux.sh
```

## جریان کار
خرید مشتری → پرداخت موفق → ووکامرس کلید لایسنس + فایل نصب را تحویل می‌دهد →
مشتری در برنامه «فعال‌سازی» را می‌زند و کلید را وارد می‌کند → برنامه با
`/wp-json/kara/v1/activate` کلید را اعتبارسنجی و دستگاه را ثبت می‌کند → باز می‌شود.

## نکات
- محدودیت دستگاه از روی **Times Activated (max)** اعمال می‌شود.
- برای ابطال یک لایسنس: در پنل وردپرس وضعیت کلید را غیرفعال کن.
- تست رایگان ۷ روزه و گارانتی بازگشت وجه ۱۴ روزه در برنامه/فروشگاه لحاظ شده.
