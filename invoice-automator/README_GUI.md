# 🧾 یارا — دستیار هوشمند کسب‌وکار (نسخه‌ی ویندوز)

برنامه‌ای با محیط گرافیکی مدرن برای ارسال پیش‌فاکتور از طریق
**پیامک (کاوه‌نگار)، واتساپ، بله و روبیکا** به‌همراه آپلود روی **وردپرس**.

اطلاعات ورود هر پیام‌رسان از داخل خود برنامه وارد می‌شود و به‌صورت
**رمزنگاری‌شده فقط روی همان کامپیوتر** ذخیره می‌گردد؛ هیچ اطلاعات شخصی
داخل کد یا مخزن گیت قرار ندارد.

---

## ✨ امکانات

- 🔐 صفحه‌ی ورود/تنظیمات جداگانه برای هر پیام‌رسان
- ✅ انتخاب اینکه از کدام پیام‌رسان‌ها استفاده شود (تیک‌دار)
- 📂 انتخاب دستی فایل PDF و ارسال هم‌زمان به همه‌ی کانال‌ها
- 🧪 دکمه‌ی «تست اتصال» برای هر سرویس
- 🌙 حالت تیره / روشن
- 🗂️ تشخیص خودکار شماره تلفن و شماره فاکتور از نام فایل

> نام‌گذاری فایل: `09122724191.pdf` یا `09122724191-1821.pdf`
> یا `مشیری-09122724191-1821.pdf` (نام-تلفن-فاکتور)

## 🚀 اجرای سریع (برای توسعه / تست)

```bash
pip install -r requirements.txt
python run_gui.py
```

## 📦 نصب روی ویندوز (`YaraPro-Setup.exe`)

نصب از طریق یک **فایل `Setup.exe` گرافیکی** انجام می‌شود که **پایتون و همه‌ی
کتابخانه‌ها داخلش باندل شده‌اند** — کاربر نهایی به هیچ پیش‌نیاز، دانلود یا
نصب اضافه‌ای نیاز ندارد (نصب کاملاً آفلاین).

**برای کاربر نهایی:** فقط `YaraPro-Setup.exe` را اجرا کنید. نصاب:
- برنامه را در `%LOCALAPPDATA%\Programs\YaraPro` نصب می‌کند
- **آیکن دسکتاپ** و میانبر منوی استارت می‌سازد
- در **«افزودن یا حذف برنامه‌ها»ی ویندوز (Uninstall)** ثبت می‌شود
- در پایان، گزینه‌ی اجرای برنامه را می‌دهد

> اگر SmartScreen هشدار داد (فایل امضای دیجیتال ندارد): More info → Run anyway.

### ساخت `Setup.exe`

نصاب با **NSIS** ساخته می‌شود و **روی لینوکس هم قابل‌ساخت است** (نیازی به
ویندوز ندارد):

```bash
sudo apt-get install -y nsis p7zip-full
bash windows-installer/build_on_linux.sh
# خروجی: windows-installer/build/YaraPro-Setup.exe
```

این اسکریپت پایتون قابل‌حمل ویندوزی (به‌همراه tkinter) را دانلود، کتابخانه‌ها
را به‌صورت wheel ویندوزی داخلش نصب، کد را کنارش می‌چیند و با NSIS یک نصاب
گرافیکی می‌سازد.

> 💡 آیکن در `assets/icon.ico` و اسکریپت نصاب در
> `windows-installer/installer.nsi` قرار دارد.

## 🗄️ محل ذخیره‌ی اطلاعات

اطلاعات ورود رمزنگاری‌شده در این مسیر ذخیره می‌شود:

```
%APPDATA%\YaraPro\settings.enc
```

کلید رمزنگاری به همین دستگاه گره خورده است، بنابراین فایل روی کامپیوتر
دیگری قابل‌استفاده نیست.

## 📁 ساختار پروژه

```
invoice-automator/
├─ run_gui.py                    ← اجرای برنامه‌ی گرافیکی
├─ assets/icon.ico               ← آیکن برنامه
├─ windows-installer/
│  ├─ build_on_linux.sh          ← ساخت Setup.exe (روی لینوکس)
│  └─ installer.nsi              ← اسکریپت نصاب گرافیکی NSIS
├─ gui/
│  ├─ app.py                     ← رابط گرافیکی اصلی
│  ├─ core.py                    ← منطق پردازش و ارسال
│  ├─ settings_store.py          ← ذخیره‌ی رمزنگاری‌شده
│  ├─ fonts.py                   ← بارگذاری فونت و آیکن
│  ├─ theme.py                   ← رنگ‌ها و تنظیمات ظاهری
│  └─ assets/fonts/              ← فونت فارسی Vazirmatn
├─ modules/                      ← ماژول‌های ارسال
├─ main.py                       ← نسخه‌ی خط فرمان قدیمی (نظارت پوشه)
└─ config.example.yaml           ← نمونه‌ی تنظیمات نسخه‌ی خط فرمان
```
