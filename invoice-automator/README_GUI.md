# 🧾 برنامه‌ی گرافیکی ارسال پیش‌فاکتور (نسخه‌ی ویندوز)

برنامه‌ای با محیط گرافیکی مدرن برای ارسال پیش‌فاکتور از طریق
**پیامک (کاوه‌نگار)، واتساپ، بله و روبیکا** به‌همراه آپلود روی **وردپرس**.

اطلاعات ورود هر پیام‌رسان از داخل خود برنامه وارد می‌شود و به‌صورت
**رمزنگاری‌شده فقط روی همان کامپیوتر** ذخیره می‌گردد؛ هیچ اطلاعات شخصی
داخل کد یا مخزن گیت قرار ندارد.

---

## ✨ امکانات

- 🔐 صفحه‌ی ورود/تنظیمات جداگانه برای هر پیام‌رسان
- ✅ انتخاب اینکه از کدام پیام‌رسان‌ها استفاده شود (تیک‌دار)
- 📂 انتخاب دستی فایل PDF و ارسال هم‌زمان به همه‌ی کانال‌ها
- 🧪 دکمه‌ی «تست اتصال» برای هر سرویس
- 🌙 حالت تیره / روشن
- 🗂️ تشخیص خودکار شماره تلفن و شماره فاکتور از نام فایل

> نام‌گذاری فایل: `09122724191.pdf` یا `09122724191-1821.pdf`
> یا `مشیری-09122724191-1821.pdf` (نام-تلفن-فاکتور)

---

## 🚀 اجرای سریع (برای توسعه / تست)

```bash
pip install -r requirements.txt
python run_gui.py
```

---

## ⚡ نصب سریع روی ویندوز (`INSTALL.bat`)

ساده‌ترین راه: فایل **`INSTALL.bat`** را دابل‌کلیک کنید. این نصب‌کننده
(مبتنی بر PowerShell، بدون نیاز به ساخت قبلی) به‌صورت خودکار:

1. اگر پایتون نباشد، خودش نصبش می‌کند
2. برنامه را در `%LOCALAPPDATA%\Programs\InvoiceAutomator` نصب می‌کند
3. **آیکن دسکتاپ** و میانبر منوی استارت می‌سازد
4. برنامه را در **«افزودن یا حذف برنامه‌ها»ی ویندوز (Uninstall)** ثبت می‌کند
5. برنامه را اجرا می‌کند

**حذف:** تنظیمات ویندوز ← Apps ← «سامانه ارسال پیش‌فاکتور» ← Uninstall.

> این روش یک فایل نصب آماده در همین بسته است و نیازی به ساخت روی ویندوز ندارد.

---

## 📦 ساخت نصب‌کننده‌ی مستقل (`Setup.exe`) — اختیاری

برای توزیع روی کامپیوترهای دیگر بدون نیاز به نصب پایتون (پایتون باندل می‌شود):

نصب برنامه از طریق یک **نصب‌کننده‌ی گرافیکی** انجام می‌شود (نه cmd). این
نصب‌کننده پس از نصب، برنامه را در **«افزودن یا حذف برنامه‌ها»ی ویندوز
(Uninstall)** ثبت می‌کند، گزینه‌ی **ساخت آیکن دسکتاپ** دارد، و **پایتون را
هم باندل می‌کند** (کاربر نهایی به هیچ پیش‌نیازی نیاز ندارد).

**ساخت نصب‌کننده** — روی **ویندوز** فایل زیر را دابل‌کلیک کنید:

```
build_installer.bat
```

این اسکریپت به‌صورت خودکار:
1. در صورت نبود، پایتون و Inno Setup را نصب می‌کند
2. برنامه را با PyInstaller (به‌همراه پایتون و آیکن) باندل می‌کند
3. با Inno Setup یک نصب‌کننده‌ی گرافیکی می‌سازد:
   `Output\InvoiceAutomator-Setup.exe`

**نصب برای کاربر نهایی:** فقط `InvoiceAutomator-Setup.exe` را اجرا می‌کند.
نصب‌کننده:
- برنامه را نصب و در لیست **Uninstall** ویندوز ثبت می‌کند
- گزینه‌ی **«ساخت آیکن روی دسکتاپ»** را نمایش می‌دهد
- میانبر منوی استارت می‌سازد
- نیازی به نصب پایتون ندارد

> 💡 آیکن برنامه در `assets/icon.ico` قرار دارد و در فایل اجرایی، آیکن
> دسکتاپ و لیست Uninstall استفاده می‌شود.

## 🗄️ محل ذخیره‌ی اطلاعات

اطلاعات ورود رمزنگاری‌شده در این مسیر ذخیره می‌شود:

```
%APPDATA%\InvoiceAutomator\settings.enc
```

کلید رمزنگاری به همین دستگاه گره خورده است، بنابراین فایل روی کامپیوتر
دیگری قابل‌استفاده نیست.

## 📁 ساختار پروژه

```
invoice-automator/
├─ run_gui.py              ← اجرای برنامه‌ی گرافیکی
├─ build_installer.bat     ← ساخت نصب‌کننده‌ی گرافیکی روی ویندوز
├─ installer.iss           ← پیکربندی Inno Setup (Uninstall + آیکن دسکتاپ)
├─ invoice_automator.spec  ← پیکربندی PyInstaller
├─ assets/icon.ico         ← آیکن برنامه
├─ gui/
│  ├─ app.py               ← رابط گرافیکی اصلی
│  ├─ core.py              ← منطق پردازش و ارسال
│  ├─ settings_store.py    ← ذخیره‌ی رمزنگاری‌شده
│  ├─ fonts.py             ← بارگذاری فونت و آیکن
│  └─ theme.py             ← رنگ‌ها و تنظیمات ظاهری
├─ gui/assets/fonts/       ← فونت فارسی Vazirmatn
├─ modules/                ← ماژول‌های ارسال (مشترک با نسخه‌ی خط فرمان)
├─ main.py                 ← نسخه‌ی خط فرمان قدیمی (نظارت خودکار پوشه)
└─ config.example.yaml     ← نمونه‌ی تنظیمات نسخه‌ی خط فرمان
```
